## ⚙️ Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Kaleb-cpu/Portfolio
```

Navigate into the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open:

```
http://localhost:3000
```
